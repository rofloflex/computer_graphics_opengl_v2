# Лабораторная работа №5. 
Подключение и работа с библиотекой ASSIMP. Настройка импорта модели.
## Описание
реализована загрузка и отображение 3D-модели с использованием библиотеки ASSIMP в среде OpenGL. Настроено подключение внешней библиотеки, выполнен импорт модели формата .obj, а также реализована базовая структура классов Model и Mesh для обработки геометрии.
